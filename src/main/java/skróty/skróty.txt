CTRL + ALT + L              ==>   Formatowanie kodu
CTRL + Q                    ==>   Dokumentacja klasy/ metody
CTRL + D                    ==>   Kopiowanie lini i wklejenie jej pod spodem
CTRL + Y                    ==>   Usuwanie bierzącej lini
sout + TAB                  ==>   System.out.println();
ALT + INSERT                ==>   Generowanie konstruktora, getterów, setterów oraz metody toString
CTRL + SHIFT + góra/dół     ==>   Przenoszenie lini w górę/ dół
